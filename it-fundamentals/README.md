# IT Fundamentals: Pre-Bootcamp Primer

## Purpose

This primer provides foundational IT knowledge for students with zero prior IT or programming experience. It prepares you for Module 01: Cloud Fundamentals of the AWS Bootcamp.

If you already have basic programming knowledge and command-line experience, you can skip directly to [Module 01: Cloud Fundamentals](../modules/01-cloud-fundamentals/README.md).

## Modules

| Module | Topics | Estimated Time | Link |
|--------|--------|---------------|------|
| 01: Computers and Operating Systems | Hardware (CPU, RAM, storage), bits/bytes, what the OS does, Linux for cloud, file systems | 18-28 min | [Module 01](modules/01-computers-and-operating-systems/README.md) |
| 02: The Command Line | Why the command line matters, navigation commands, file operations, searching | 21-29 min | [Module 02](modules/02-command-line/README.md) |
| 03: Networking and the Internet | TCP/IP model, IP addresses, DNS, ports, TCP vs UDP, HTTP request/response, status codes, HTTPS | 28-37 min | [Module 03](modules/03-networking-and-internet/README.md) |
| 04: APIs and Programming Basics | REST APIs, JSON, variables, functions, loops, conditionals | 30-41 min | [Module 04](modules/04-apis-and-programming/README.md) |
| 05: Version Control with Git | Why Git matters, core concepts, essential commands | 21-29 min | [Module 05](modules/05-version-control-git/README.md) |
| 06: Security Fundamentals | Authentication vs authorization, encryption, principle of least privilege | 13-21 min | [Module 06](modules/06-security-fundamentals/README.md) |

**Total estimated self-study time:** 2 to 3 hours

## Prerequisites

- A computer with internet access (Windows, macOS, or Linux)
- No prior IT or programming experience required

**Required software:**

| Tool | Windows | macOS | Linux |
|------|---------|-------|-------|
| Terminal | Git Bash (bundled with Git) or WSL | Terminal.app (pre-installed) | Terminal (pre-installed) |
| Git | Download from [git-scm.com](https://git-scm.com/download/win) | `xcode-select --install` or download from [git-scm.com](https://git-scm.com/download/mac) | `sudo apt install git` |
| Python 3 | Download from [python.org](https://www.python.org/downloads/) | `brew install python3` or download from [python.org](https://www.python.org/downloads/) | `sudo apt install python3` |
| Text editor | VS Code (recommended) | VS Code (recommended) | VS Code (recommended) |

> **Tip:** On Windows, use Git Bash or Windows Subsystem for Linux (WSL) for command consistency with macOS and Linux.

## Recommended Learning Path

Complete the modules in order. Each module builds on skills from the previous one:

1. [Module 01: Computers and Operating Systems](modules/01-computers-and-operating-systems/README.md) (no prerequisites)
2. [Module 02: The Command Line](modules/02-command-line/README.md) (no prerequisites)
3. [Module 03: Networking and the Internet](modules/03-networking-and-internet/README.md) (Module 02 recommended)
4. [Module 04: APIs and Programming Basics](modules/04-apis-and-programming/README.md) (Module 02 recommended)
5. [Module 05: Version Control with Git](modules/05-version-control-git/README.md) (Module 02 recommended)
6. [Module 06: Security Fundamentals](modules/06-security-fundamentals/README.md) (Module 02 recommended)

After completing all 6 modules, proceed to [Module 01: Cloud Fundamentals](../modules/01-cloud-fundamentals/README.md) to begin the AWS Bootcamp.
