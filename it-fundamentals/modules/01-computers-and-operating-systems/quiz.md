# IT Fundamentals Module 01: Quiz

1. What is the primary purpose of RAM (Random Access Memory) in a computer?

   A) Store data permanently, even when the computer is turned off
   B) Execute instructions and perform calculations
   C) Store data temporarily while programs are running
   D) Connect the computer to a network

2. True or False: Linux is the dominant operating system in cloud computing, with the majority of cloud servers, containers, and serverless environments running on Linux.

3. Which of the following is NOT a responsibility of an operating system?

   A) Allocating CPU time to running programs
   B) Managing memory across applications
   C) Generating electricity for the hardware
   D) Organizing files into directories

---

<details>
<summary>Answer Key</summary>

1. **C) Store data temporarily while programs are running**
   RAM holds data that the CPU needs to access quickly while programs are running. When you close a program or shut down the computer, the data in RAM is cleared. Storage (SSD or HDD) is used for permanent data, and the CPU executes instructions.
   Further reading: [What Is RAM? (Crucial)](https://www.crucial.com/articles/about-memory/support-what-does-computer-memory-do)

2. **True.**
   Linux is the most widely used operating system in cloud computing. The majority of cloud servers, containers, and serverless environments run on Linux distributions. Its open-source nature, stability, and flexibility make it the preferred choice for server workloads.
   Further reading: [What Is Linux? (Red Hat)](https://www.redhat.com/en/topics/linux/what-is-linux)

3. **C) Generating electricity for the hardware**
   An operating system manages CPU time, memory, storage, and networking, and it provides a user interface. It does not generate electricity. Power supply is a hardware function, not a software function. The OS manages how hardware resources are allocated, but it does not produce the physical energy those components consume.
   Further reading: [The Linux Kernel Documentation](https://www.kernel.org/doc/html/latest/)
