# IT Fundamentals Module 02: Quiz

1. What does the command `cd ..` do in a terminal?

   A) Changes to the home directory
   B) Creates a new directory
   C) Moves up one directory in the file system
   D) Lists the contents of the current directory

2. Which command displays the contents of a file in the terminal?

   A) `ls`
   B) `cat`
   C) `mkdir`
   D) `mv`

---

<details>
<summary>Answer Key</summary>

1. **C) Moves up one directory in the file system**
   The `cd` command stands for "change directory." The `..` notation refers to the parent directory, so `cd ..` moves you one level up in the directory hierarchy. For example, if you are in `/home/student/projects`, running `cd ..` takes you to `/home/student`.
   Further reading: [GNU Coreutils: cd](https://www.gnu.org/software/coreutils/manual/coreutils.html)

2. **B) `cat`**
   The `cat` command (short for "concatenate") reads a file and prints its contents to the terminal. `ls` lists directory contents, `mkdir` creates a new directory, and `mv` moves or renames a file.
   Further reading: [GNU Coreutils: cat](https://www.gnu.org/software/coreutils/manual/coreutils.html)
