# IT Fundamentals Module 04: Quiz

1. True or False: REST APIs use standard HTTP methods (GET, POST, PUT, DELETE) to perform actions on resources.

2. In your own words, what is a variable in programming?

3. Which of the following is valid JSON?

   A) `{name: "Jane"}`
   B) `{"name": "Jane",}`
   C) `{"name": "Jane"}`
   D) `{'name': 'Jane'}`

---

<details>
<summary>Answer Key</summary>

1. **True.**
   REST (Representational State Transfer) APIs use standard HTTP methods to perform operations on resources: GET to retrieve data, POST to create new data, PUT to update existing data, and DELETE to remove data. This makes REST APIs predictable and consistent.
   Further reading: [REST (MDN Web Docs)](https://developer.mozilla.org/en-US/docs/Glossary/REST)

2. **Sample answer:** A variable is a named container that stores a value in a program. The value can be a number, text, a boolean, or other data types, and it can be changed as the program runs. For example, `name = "Jane"` creates a variable called `name` that holds the text "Jane."
   Further reading: [Python Data Model (python.org)](https://docs.python.org/3/reference/datamodel.html)

3. **C) `{"name": "Jane"}`**
   Valid JSON requires keys to be strings in double quotes, not unquoted (A) or in single quotes (D). Trailing commas are not allowed in JSON (B). Option C follows all JSON syntax rules: the key is a double-quoted string, the value is a double-quoted string, and there is no trailing comma.
   Further reading: [JSON (MDN Web Docs)](https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Objects/JSON)

</details>
