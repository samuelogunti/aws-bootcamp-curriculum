# Tech Stack

## Project Type
Markdown-based educational content repository. No application code — this is a curriculum.

## Content Format
- All content written in Markdown (`.md` files)
- Diagrams described in text or referenced as external images
- Code examples embedded in fenced code blocks with language tags

## AWS Services Covered
EC2, S3, RDS, DynamoDB, Lambda, API Gateway, ECS, Fargate, VPC, IAM, CloudFormation, SAM, CDK, CloudWatch, X-Ray, Route 53, ALB, SQS, SNS, EventBridge, Step Functions, CloudFront, ElastiCache, KMS, Secrets Manager, WAF, Shield, CloudTrail, Config, GuardDuty, Athena, Kinesis, Organizations, Control Tower

## Tools Students Will Use
- AWS Management Console
- AWS CLI
- AWS SAM CLI
- Docker
- Git
