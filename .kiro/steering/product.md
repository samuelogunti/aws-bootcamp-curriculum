# Product Overview

## Project
AWS Bootcamp: From Novice to Architect — a 20-module instructor-led curriculum.

## Audience
Beginner to intermediate developers with basic programming knowledge but no prior AWS or cloud experience. By the end, students should be able to independently design, build, and operate production-grade AWS architectures.

## Content Philosophy
- Progressive complexity: each module builds on the previous
- Hands-on first: every concept is reinforced with a lab exercise
- Official sources only: all references must link to AWS documentation
- Instructor-led: content is written for a teacher to deliver, not self-paced

## Curriculum Phases
1. **Cloud Foundations** (Modules 01–03): Cloud concepts, IAM, VPC
2. **Core Services** (Modules 04–08): Compute, storage, databases, networking, messaging
3. **Building Applications** (Modules 09–12): Serverless, containers, IaC, CI/CD
4. **Production Readiness** (Modules 13–16): Security, monitoring, cost, reliability
5. **Architecting** (Modules 17–20): Well-Architected, patterns, advanced topics, capstone
