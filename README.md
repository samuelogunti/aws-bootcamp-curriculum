# AWS Bootcamp: From Novice to Architect

A comprehensive, hands-on AWS bootcamp curriculum designed to take students from zero cloud experience to confidently designing and building production-grade architectures.

## Prerequisites

- Basic programming knowledge (any language)
- A computer with a web browser
- An AWS account (Free Tier eligible)

## Curriculum Structure

| Phase | Modules | Focus |
|-------|---------|-------|
| 1 — Cloud Foundations | 01–03 | What is cloud, AWS console, core concepts |
| 2 — Core Services | 04–08 | Compute, storage, databases, networking |
| 3 — Building Applications | 09–12 | Serverless, containers, CI/CD, IaC |
| 4 — Production Readiness | 13–16 | Security, monitoring, cost, reliability |
| 5 — Architecting | 17–20 | Design patterns, Well-Architected, capstone |

## How to Use This Repo

Each module folder contains:
- `README.md` — Lesson content, concepts, and diagrams
- `lab/` — Step-by-step hands-on exercises
- `quiz.md` — Knowledge check questions
- `resources.md` — Further reading and official docs links

## Estimated Duration

- Full bootcamp: 10–12 weeks (2 modules/week)
- Each module: 2–3 hours (lecture + lab)
